# comict(コミクト)

## 概要
自分の好みの漫画になかなか出会うことができない...</br>
1巻だけ買ったけどなんか違った...</br>
こんな悩みを解決するために生まれたプロダクト</br>
アンケートに答えることで、ユーザーの属性を判別し、漫画をレコメンドします。</br>
また、趣味や嗜好が似ているユーザーとのマッチング機能もあるため、似たような漫画が好きなユーザーと繋がることができます。</br>
